# 快速游玩流程指南

规则指引书里没说怎么办？

即使你已经玩过很多次《调查 : 深入》，即使我们已经尽力让这份《规则指引书》尽善尽美，有时也会遇到规则指引书中未明确规定的情况。请不要过于纠结，和大家商量达成共识后继续游戏。

对于首次体验《调查 : 深入》的玩家，我们建议先以标注了Vol.1脚标的卡牌组成卡组，进行游戏。

当你熟悉了游戏流程，将标注了Vol.2脚标的卡牌加入各个卡组，以获得完整体验。

- 「指挥者」确认组队情况，选择「调查员」，随机分配「调查顺位」

- 「指挥者」抽取(4)张初始手牌

- 开始一「轮」游戏

- 触发一张新的「环境」卡牌

- 「调查员」依照「调查顺位」轮流进行「操作阶段」。

每位「调查员」有独立的「操作阶段」，「指挥者」可能操作多次。

- 「操作阶段」开始：「调查员」需要做的事（特定顺序）：

- 触发「环境」效果、「个体机制」或「调查员」技能。

- 从「策略卡组」顶部抽(1)张牌，加入手牌。

- 进入「操作阶段」：「调查员」可以做的事（任意顺序）：

- 对「情报卡组」进行调查。

- 使用自己手牌中的「策略」卡牌（消耗「情报」卡牌）。

- 触发「个体机制」。

- 使用「调查员」的技能。

- 「操作阶段」结束：「调查员」需要做的事（特定顺序）：

- 触发「环境」效果、「个体机制」或「调查员」技能。

- 「调查员」的「指挥者」需要将手牌弃置到 ≤ (8)张。

- 【禁忌真相】被调查，（或基于其他「环境」效果本「轮」结束）。

结算「SAN」消耗，收回场上所有「情报」卡牌，开始下一「轮」游戏。回到步骤3。

- 仅剩一个队伍「调查员」存活，胜负已定。

# 故事背景

## 「调查工会」

欢迎回来，这里是「异常联合调查工会」（Investigation Guild of Anomaly），通常被简称为「调查工会」。我们是一个跨国独立组织，成员围绕一些基本的规则，各自在自己的行动范围内执行任务。

过去几十年间异常「环境」的识别案例激增，「调查工会」的业务也到了前所未有的新高峰。我们的客户一般都有他们各自的盘算，不论是尝试控制舆情的地方政府，还是那些疯狂的异教徒，只要风险可靠收益到位，我们就可以是他们一线任务的践行者。

## 「指挥者」

你是「调查工会」的一名「指挥者」，能有这样的身份，说明你已经熬过了最困难的头几年，不仅活了下来，而且还神志清醒地坚持留在了这里。坐到这个位置，你已经不需要实地执行任务，你的工作就是为你的「调查员」在任务过程中下发「策略」指令，通过经验、直觉和智慧引领他们完成任务。

我是你的新副官，受限于「调查工会」的安全协议，我只能以文字形式和你沟通，但这不会影响我为你提供协助。我们花了很多精力才把你从先前的事故中解救出来，你可能正经历一些糟糕的记忆问题，但任务不等人，让我帮你快速介绍一下情况，好让你进入状态。

## 「调查任务」

每次任务中，你都需要应对不同的异常「环境」，你将会揭示其中的【禁忌真相】所在。「情报」虽然是多多益善，但调查到【禁忌真相】却十分危险，频繁调查到【禁忌真相】的「调查员」往往都会「陷入疯狂」，不过这里毕竟是「调查工会」，我想这也是大家都能理解的结局😞。

若你指挥的「调查员」都「陷入疯狂」，你就必须放弃这次调查，因此为了任务成功，务必还是谨慎行事。

调查过程中，你会遇到的阻碍多种多样，但同行往往是最为危险的。你要确保自己的队伍能完成任务、全身而退，因此一旦进入任务区域，务必不要手软😡。

现在，我们需要你立刻投身下一个任务，切记仔细阅读《规则指引书》，里面包含了你所有需要知道的信息。

祝你好运，调查开始了。

# 游戏总览

在《调查 : 深入》的游戏中，每名玩家扮演一位「指挥者」，指挥「调查员」对「情报卡组」进行调查，尝试了解【禁忌真相】在「情报卡组」中的位置，让自己的「调查员」规避它，并让其他队伍的「调查员」调查到【禁忌真相】。

你可能与其他「指挥者」组队作战（4、6名玩家时），又或者独自面对威胁（2、3名玩家时）。每一「轮」游戏都伴随与众不同的「环境」卡牌开展，给玩家们带来新的威胁（又或者机遇？）。

记住，一「轮」游戏的胜负结果只是暂时的，只有让自己的「调查员」生存到最后的「指挥者」才能迎接真正的胜利。

# 游戏目标

「调查员」调查到【禁忌真相】会损失「SAN」，「SAN」消耗完的「调查员」会「陷入疯狂」。

你要通过「情报」卡牌获取关于【禁忌真相】在「情报卡组」中位置的信息，巧妙地使用「策略」和「情报」卡牌，运用你「调查员」的技能，利用与众不同的「环境」与敌对「指挥者」对抗。

当除你以外的队伍所指挥的「调查员」均「陷入疯狂」时，你（和你的队友）就获得了本局游戏的胜利！

# 游戏准备

#### 准备卡组

- 将「策略」、「情报」、「环境」、「辅助」卡牌各自洗切整理成一堆，分别组成(4)组卡组。

#### 确定「指挥者」人数

- 《调查 : 深入》支持2~6名玩家游玩。若是组队对战，同一队伍的各位「指挥者」会共享「情报区」的「情报」卡牌（使用或消耗），共享获得的信息（手牌或「情报」相关），但只能指挥自己的「调查员」、使用自己的手牌。

- 原则上，同一队伍的信息共享仅限于窥探的信息，队友间不可互相检视手牌，沟通信息需公开对话。但玩家们可以按照实际游戏情况，选择合适的信息共享规则。

#### 分配「调查员」

- 每位「指挥者」会被派发(2 × N)张候选的「调查员」卡牌，从中选取指挥(N)位「调查员」，建议以如下配比进行游戏：

- 2位玩家，分别在2个队伍；每人指挥2位「调查员」

- 3位玩家，分别在3个队伍；每人指挥2位「调查员」

- 4位玩家，分别在2个队伍；每人指挥1位「调查员」

- 6位玩家，分别在2个或3个队伍；每人指挥1位「调查员」

- 玩家也可以其他配比开始游戏，场上开局时调查员数量需≥4。

#### 分配「调查顺位」

- 将参与本局游戏的「调查员指示卡」洗切，随机布置到场上的「顺位指示卡」边。

#### 记录「SAN」

- 卡牌右上角的骰子点数，标识该「调查员」的「SAN」上限。

- 将(1)个骰子以「SAN」上限的点数朝上，放到该处，用以记录该「调查员」的当前「SAN」。多余的骰子可以放置到一边待用。

#### 抽取初始手牌

- 各位「指挥者」需要按「调查员」的「调查顺位」，依次从「策略卡组」抽取(4)张「策略」卡牌纳入手牌。

- 确认完成以上步骤后，你们可以参照下图“场地布置”，设置你们的桌面（以6位玩家，分属2个队伍为例），或选择适合你们的布局。

# 开始游戏

每局游戏开始后，将以多「轮」游戏进行推进。

## 触发「环境」卡牌

每「轮」游戏开始前，从「环境卡组」顶部抽取(1)张「环境」卡牌，该卡牌卡面效果在本「轮」内会对所有「调查员」持续生效。

若本「轮」并非第一「轮」游戏，这张新的「环境」卡牌会代替上一轮的「环境」卡牌生效（上一「轮」的「环境」卡牌不洗回「环境卡组」）。

## 新一「轮」游戏开始

每「轮」游戏由上一「轮」调查到【禁忌真相】的「调查员」先执行「操作阶段」。之后，玩家需要依照「调查顺位」指挥各自的「调查员」，依次执行「操作阶段」。

如果这是第一「轮」游戏，第(1)顺位的「调查员」先执行「操作阶段」。

## 轮流进行「操作阶段」

每一「轮」游戏包含不限数量的「操作阶段」，每位「指挥者」指挥对应「调查员」在其「操作阶段」内行动。「操作阶段」分为以下3个步骤：

#### 开始「操作阶段」：「调查员」需要依照顺序逐一执行下列操作：

- 判定「环境」、已经对「调查员」布置的「个体机制」或「调查员」技能，是否有可以/必须触发生效的，如有，触发卡牌或技能。

- 从「策略卡组」顶部抽(1)张牌，加入手牌。

#### 进入「操作阶段」：「调查员」可以任意顺序随意执行下列操作：

- 对「情报卡组」进行调查（从「情报卡组」顶部，顺序翻看并公示(1)~(3)张「情报」卡牌，然后纳入自己的「情报区」）。如无特殊描述，每位「调查员」每个「操作阶段」必须、且只能执行一次调查。

- 使用「情报区」的「情报」卡牌，执行其卡面效果。该行动可以不限次数自由执行。

- 使用自己手牌中的「策略」卡牌（通常需要消耗「情报区」的「情报」卡牌），该行动可以不限次数自由执行。

- 触发可以或必须在「操作阶段」使用的的「个体机制」，如无特殊说明，可以不限数量使用所有布置在当前「调查员」身上的「个体机制」。

- 使用「调查员」的技能，如无特殊说明，可以不限次数使用任意技能。

### 「操作阶段」结束时，「调查员」需要依照顺序逐一执行下列操作：

- 判定「环境」卡牌、已经对「调查员」布置的「个体机制」，或「调查员」技能是否有可以/必须触发生效的，如有，触发卡牌或技能。

- 「调查员」的「指挥者」需要将手牌弃置到 ≤ (8)张。

## 结束一「轮」游戏

任意「调查员」调查到【禁忌真相】时，本「轮」游戏结束。该「调查员」的「SAN」将相应减少（无特殊效果时，减少(1)点「SAN」）。

收回场上所有的「情报」卡牌（包括未调查的「情报卡组」剩余部分、「情报弃牌区」和「情报区」的「情报」卡牌）洗切组成新的「情报卡组」。

各位「指挥者」的手牌、场上已布置的「个体机制」均不变。此后，下一「轮」游戏开始。

### 「SAN」归零的「调查员」

- 若「调查员」因此导致「SAN」归零，其将会「陷入疯狂」。将其「调查员」卡牌与「调查员指示卡」翻面，所有对其布置的「个体机制」被弃置。

- 「陷入疯狂」的「调查员」没有「操作阶段」，其调查顺位在所有判定中被自然跳过（如：下一顺位的「调查员」「陷入疯狂」，则针对下一顺位的效果将顺延到下下顺位）。

- 「陷入疯狂」的「调查员」具有额外的【禁忌知识】和【疯狂蔓延】技能，如无特殊说明，在「调查员」「陷入疯狂」时立即触发。

- 如无特殊说明，「陷入疯狂」的「调查员」无法成为卡牌的指定对象、也不受卡牌或技能的效果影响。

- 由「调查员」的【疯狂蔓延】技能带来的「辅助」卡牌，依旧可以受到其他「策略」卡牌的影响，但需要检查其卡面文本。

- 「陷入疯狂」的「调查员」若通过任意手段恢复「SAN」，其将会「恢复清醒」并拥有恢复点数的「SAN」，没有「SAN」的特殊「调查员」无法用此手段「恢复清醒」。

# 胜负判定

其他队伍的所有「调查员」都「陷入疯狂」时，最后一只队伍及其「指挥者」获得游戏胜利。

若出现一个操作导致所有剩余「调查员」都「陷入疯狂」的情况，最后「陷入疯狂」的「调查员」及其所在队伍获得游戏胜利。
