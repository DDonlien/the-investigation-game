# 调查附录

# 特殊文本标识

游戏的文本中存在3种特殊文本标识：

- 方括号如【胁迫 · I】表示卡牌 (包括「调查员」) 名称，其中【禁忌真相】有特殊的颜色标识。

- 直角引号如「调查员 」表示特有的概念词汇，其中「SAN」、「陷入疯狂」有特殊的颜色标识。

- 斜体加粗字样如“盲从”表示一系列特殊行动。特殊行动通常会在卡面写明效果，同时也会在此一一列出。有时会出现特殊行动的用词与常规文本用词重复的情况，此时请注意观察是否是斜体加粗字样，仅斜体加粗字样指代特殊行动。其中调查有特殊的颜色标识。

- 由于部分字样可能在更早的版本被作为普通词汇，而在后续版本被视作特殊行动，因此可能出现卡面并未进行正确斜体加粗的情况。针对此类情况，会在下列特殊行动列表中标识出首次追加该行动的补充包版本，以及受影响的早期版本卡牌。

- 未被列出的早期版本卡牌，即使涉及到相同文本，依旧不被视作特殊行动。

# 特殊行动列表

### 调查

从「情报卡组」顶部，顺序翻看并公示(1)~(3)张「情报」卡牌，然后纳入自己的「情报区」。如无特殊描述，每位「调查员」每个「操作阶段」必须、且只能执行一次调查。调查有特殊的颜色标识。

### 窥探

私下（队内）顺序查看(N)张「情报」卡牌，并将其原样放回。

### 调整

将卡牌放回对应「卡组」的最上、最下、或其原位。若调整的对象来自于「弃牌区」，可以将其放回其所属「卡组」的最上、最下，或其在「弃牌区」中的原位。

### 跳过

你在当前的「操作阶段」，无需且不能调查。跳过可以在「操作阶段」的任意节点生效，其效果持续到「操作阶段」结束。

### 反转

反转当前的调查顺序，将所有「顺位指示卡」翻面。

### 胁迫

调查的「情报」卡牌数量 + (N)，N取决于胁迫后的数字。例如，当「调查员」受到胁迫 + (2)，则在没有其他效果的情况下，其调查范围从(1) ~ (3)变为(3) ~ (5)。

胁迫会在「调查员」执行调查时生效。所有对同一位「调查员」布置的胁迫，都会在其最近一次调查中累加生效。若「调查员」没有进行调查，胁迫视为未生效。

### 盲从

「调查员」的「操作阶段」开始后，必须首先进行调查。

盲从会在被布置的「调查员」的「操作阶段」开始后生效，效果持续到「调查员」执行调查。若「调查员」使其无需执行当前「操作阶段」或当前「操作阶段」的调查（如提前被布置【跳过】或【异时癖】），盲从视为未生效。

### 回收

将卡牌移至其上一个所在位置，如无特殊说明，应当对上一个所在位置的卡牌进行洗切。

通过技能直接触发、激活的「辅助」卡牌，其上一个所在位置为「辅助卡组」而非手牌，因为它们并非从手牌中打出。

于Vol.3首次加入游戏。Vol.1、Vol.2的部分卡牌并未标识，受影响的卡牌如下

- 「调查员」【仿生人：OP3-C】

## 《卡牌统计表》阅读引导

### 包含版本：

- Vol.1、Vol.2是首次众筹发行包含的标准卡牌

- Vol.3是第一个补充内容，部分卡牌在首次众筹发行过程中解锁

- Vol.Special为特殊情况发行的卡牌

### 说明更新版本：

指该卡牌效果、判断与对应说明书版本的匹配关系。由于印刷文件的固定性，该机制用于弥补不可回溯的bug与补充机制。v1.0版本对应首次众筹发行的卡牌印刷版本。

如：

当前最新版说明书版本为v1.2，

某张卡牌【A】的“说明更新版本”为v1.0，意味着该卡牌最后一次用法更新在v1.0版本发生，且该效果维持到v1.2继续适用；

另一张卡牌【B】的“说明更新版本”为v1.2，意味着该卡牌在v1.2出现了效果或判例更新，请参照[[调查深入-规则指引书|最新版规则指引书]]以及卡牌效果说明进行游戏判定。
