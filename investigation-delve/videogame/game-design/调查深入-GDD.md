# 概念

「调查 : 深入」的电子游戏版本，包含几个部分

# 一阶段

电子化的调查深入，包含单人和p2p联机模式，玩法和调查深入桌游完全一致（可能扩充内容）

# 二阶段

《杀戮尖塔》版本的调查深入，玩法不变，但增加了dbg和外循环部分

# 三阶段

复杂化的dbg玩法，参考《暗黑地牢》

玩家在调查工会选择调查员。

不同的调查员会附带不同的预选卡组。

角色

指挥者是玩家角色

指挥者包含以下几个属性：

调查员是受玩家指挥的游戏中角色

调查员包含以下几个属性：

- 可选策略：

- 携带该调查员后，在随机卡池会解锁的策略卡牌

- 列表可升级解锁，初始值随机（取决于调查员）

- 预选策略：

- 携带该调查员后，可以在“可选策略”中预选策略卡牌带入初始游戏

- 数量可升级，标准 = 5

- SAN：

- 调查员的理智值

- 数值可升级，标准 = 3

控制

- 简单的鼠标点击控制，参考《暗黑地牢》

镜头

- 简单的固定镜头，参考《暗黑地牢》
